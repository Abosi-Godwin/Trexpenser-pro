# Trexpenser — Web App

The core financial management application. Where users track expenses, manage budgets, and get clarity on their finances.

🔗 **Live:** [trexpenser-dashboard.vercel.app](https://trexpenser-dashboard.vercel.app)

---

## Tech Stack

- **Library:** React 18
- **Language:** JavaScript
- **Styling:** Tailwind CSS
- **Auth & Database:** Supabase
- **Data Fetching:** TanStack Query v5
- **Forms:** React Hook Form
- **Routing:** React Router v6

## Features

- Email/Password authentication + Google OAuth
- Expense tracking with categories
- Budget management
- Financial overview dashboard
- Dark mode support

## Getting Started

```bash
git clone https://github.com/Abosi-Godwin/trexpenser-dashboard.git
cd trexpenser-dashboard
npm install
```

Create a `.env.local` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

```bash
npm run dev
```
